# Student Platform

## Overview

Student Platform is a full-stack web application designed to foster collaborative learning among students. The platform allows learners to ask academic questions, share knowledge through discussion posts, and engage with fellow students in a centralized online community. An administrative dashboard enables efficient management of users, posts, and platform activities.

## Features

- User registration and secure authentication
- Student profile management
- Create and share discussion posts
- Ask and answer academic questions
- Search posts by module, user, or keyword
- View recent discussions
- Upload images with posts
- Admin dashboard for managing users and content
- Responsive and user-friendly interface

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap
- XAMPP

## Installation

1. Clone the repository.
2. Import the MySQL database.
3. Configure the database connection.
4. Run the project using XAMPP or another PHP server.
5. Open the application in your browser.
